Help

What are Variants?

Variants are different versions of the same product. For example, a t-shirt might have variants for different colors and sizes.

SKU Guidelines

Create unique SKUs for each variant. Examples:

  • BIKE-RED-M
  • HELMET-BLK-L
  • TIRE-700C-25
